# RFID Attendance & Activity Management System

## 📌 Description
A full-stack web application for managing attendance and activities using RFID technology.

This system allows users to check in via RFID cards while administrators can manage users, activities, locations, and view reports through a dashboard.

## 🚀 Features
- RFID-based check-in system
- User management (Add / Edit / Delete)
- Activity & checkpoint management
- Admin dashboard with charts and statistics
- Location tracking (Map integration)
- Export attendance data to CSV
- Authentication system (Login / Logout)

## 🛠 Tech Stack
- Backend: PHP
- Database: MySQL
- Frontend: HTML, CSS, JavaScript
- Charts: Chart.js
- Other: RFID integration

## 📂 Project Structure
- `/api` → API endpoints
- `/images` → images and assets
- `/uploads` → uploaded files
- Core files → main system logic
